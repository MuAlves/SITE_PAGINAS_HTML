## Regras para todas as páginas HTML

- Utilizar a estrutura semântica do HTML5 com as tags obrigatórias:
  - `<nav>` — Menu de navegação
  - `<main>` — Conteúdo principal
  - `<section>` — Agrupamento de conteúdos
  - `<footer>` — Rodapé da página

- O conteúdo das tags `<nav>` e `<footer>` deve ser **idêntico em todas as páginas**.

- O menu de navegação (`<nav>`) deve conter **3 links** que direcionem para:
  - `index.html` (Início)
  - `tutorialCSS.html` (Tutorial CSS)
  - `framework.html` (Framework)

- Todas as regras de estilo CSS devem estar em arquivo externo `estilo.css`.

- Cada página deve conter:
  - Um título com seu **nome completo** em uma tag `<h1>`
  - Pelo menos um parágrafo (`<p>`)
  - Rodapé contendo seu **RGM** e sua **turma**
